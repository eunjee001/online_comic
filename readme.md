# 👩‍🎨웹툰 앱
## 🛠️기능
- webView 사용
- ViewPager2
  - fragment와 사용
  - tablayoutMediator를 통해 tabLayout 과 함께 쓰기
- Fragment
- SharedPreference
- Dialog




## 🛠️구현 기능 
- ViewPager2를 이용해 N개의 Fragment를 구성
- 각 Fragment는 webView를 전체화면으로 구성
- TabLayout과 ViewPager2를 연동하고, Tab이름을 동적으로 바꿀 수 있음
- 웹툰의 마지막 조회 시점을 로컬에 저장하고, 앱 실행 시 불러옴
